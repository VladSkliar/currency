# Currency

Technology:
Django/Django Rest Framework,
SQLLite3(PostgreSQL on deploy)

### Installation

Just clone repo and change directory. Than open your virtual environment and install requirements package:
```sh
$ pip install -r requirements.txt
```
Than run in console:
```sh
$ ./manage.py migrate
$ ./manage.py runserver 
```
